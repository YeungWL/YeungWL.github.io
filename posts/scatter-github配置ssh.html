    ssh-keygen -t rsa -C 'xxx@xxx.com' 然后一路回车(-C 参数是你的邮箱地址)

    # 2. 然后会出现：Enter file in which to save the key (/Users/yzq/.ssh/id_rsa):

输入：n  （不覆盖）
